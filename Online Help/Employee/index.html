Login

To login to the system, the user is required to enter a valid username and password on the Login Form at the center of the Main Page (See Figure 1-1).

Figure 1-1 DOST-CO HRMIS Main Page

How to Login

1. Enter Username and Password on the entry boxes labeled respectively. Then click Login button (See Figure 1-2).

Figure 1-2 Login Form

2. If login is successful, you will be directed to the Default Screen.

Default Screen

Once logged in to the system you will be automatically directed to the Notification Section (See Figure 2).

Figure 2 Default Page

Employee Module

The Employee Module is one of DOST-CO HRMIS component that allows employees to view their respective personnel data, compensation details, attendance records, file requests and status of their requests.


The Employee Module allows employees to:

  • View Personal Information
  • View Family Background
  • View Education
  • View Examination
  • View Work Experience
  • View Voluntary Works
  • View Trainings
  • View Other Information
  • View Position Details
  • View Duties and Responsibilities
  • View Documents
  • View ARE
  • View and Print Personal Data Sheet

  • View Daily Time Record
  • View Leave Balance

  • File Requests On-line
  • View and Print Requests

  • View Notifications on Employee Requests
  • Cancel Requests

  • View Personnel Profile
  • View Income
  • View Deduction Summary
  • View Premiums and Loans
  • View Remittances
  • View and Print Reports

  • Logout from the system

Attendance

This part of the section contains information regarding the employee’s attendance details. Information such as the employee’s time in and time out, the total number of working days, the number of days the employee reported to office and the number of days the employee was absent or was on leave.

1. To access the Daily Time Record, click the Attendance Tab (See Figure 3-16).

Figure 3-16 Attendance page

Note: Default DTR shown is of the current month.

2. To view attendance from previous month(s), specify Month and Year desired from the drop-down lists.

3. When done, click Search button to display the DTR that matched the parameters chosen.

How to Print Daily Time Record

1. To print a copy of DTR, click Print/Preview button . The printable document will appear in a new window.

Figure 3-16 Attendance page

Request

This section allows the employees to send request files on-line. This is where the employee can file for Official Business Request, file for Seminars/Trainings attendance, file for Leave application, file for Travel Order authority, update employee’s Personal Data Sheet, request for reports generation and monetization of leave benefits.

1. To access the Request tab, click the Request tab (See Figure 3-17).

Figure 3-17 Request page

Official Business

1. Click the Official Business tab from the dropdown list of Request tab to view the Official Business page (See Figure 3-18).

Figure 3-18 Official Businiess page

2. Type in the Date, Time In, Time Out, Destination and Purpose.

3. When done, click Submit button to proceed to User Identity Validation or the Cancel button to cancel operation.

Official Business

1. Click the Official Business tab from the dropdown list of Request tab to view the Official Business page (See Figure 3-18).

Figure 3-18 Official Businiess page

2. Type in the Date, Time In, Time Out, Destination and Purpose.

3. When done, click Submit button to proceed to User Identity Validation or the Cancel button to cancel operation.

Leave

1. Click the Leave tab from the dropdown list of Request tab to view the Leave page (See Figure 3-19).

Figure 3-19 Leave page

2. Specify the Type of Leave from the drop-down list.

3. Type in the Date, Number of Days Applied and Reasons for Applying.

4. When done, click Submit button to proceed to User Identity Validation or the Cancel button to cancel operation.

Travel Order

1. Click the Travel Order tab from the dropdown list of Request tab to view the Travel Order page (See Figure 3-20).

Figure 3-20 Travel Order page

2. Type in the Destination, Date and Purpose.

3. When done, click Submit button to proceed to User Identity Validation or the Cancel button to cancel operation.

PDS Update

1. Click the PDS Update tab from the dropdown list of Request tab to view the PDS Update page (See Figure 3-21).

Figure 3-21 PDS Update page

2. To update PDS, specify the Type of Profile from the drop-down list.

3. To update previous information, click the Edit button opposite the information to edit.

4. To add new information, fill-up the entry box that constitutes the parameters for adding new information.

5. When done, click Submit button to proceed to User Identity Validation or the Cancel button to cancel operation.

Leave Monetization

1. Click the Leave Monetization tab from the dropdown list of Request tab to view the Leave Monetization page (See Figure 3-22).

Figure 3-22 Leave Monetization page

2. Place a tick mark on the Commutation checkbox to request for Leave Commutation.

3. When done, click Submitbutton to file request for Leave Monetization.

Note: Only regular employees can file for Leave Monetization.
Note: Check leave credits and review the guidelines before submitting request to avoid errors and conflict.

DTR Update

1. Click the DTR Update tab from the dropdown list of Request tab to view the DTR Update page (See Figure 3-23).

Figure 3-23 DTR Update page

2. To add new information, fill-up the entry box that constitutes the parameters for adding new information including Date, Time, Reason and Supporting Evidence.

3. Specify Authorized Official (Signatory) from the dropdown list.

5. When done, click Submit button to proceed to User Identity Validation or the Cancel button to cancel operation.

Compesary Time Off

1. Click the Compesary Time Off tab from the dropdown list of Request tab to view the Compesary Time Off page (See Figure 3-24).

Figure 3-24 Compesary Time Off page

2. To add new information, fill-up the entry box that constitutes the parameters for adding new information including Date, Time and Purpose.

3. Specify Recommending Approval and Approval / Disapproval from the dropdown list.

5. When done, click Submit button to proceed to User Identity Validation or the Cancel button to cancel operation.

Notification

This part of the section allows the employee to view the status of requests filed, as well as the history of employee’s requests.

1. To access the Notification page, click the Notification bell on navigation bar (See Figure 3-25).

Figure 3-25 Notification page

2. To view the status and history of requests, click View button. A new window will open showing the history of requests.

3. To cancel a request, click the Cancel button opposite the corresponding request to cancel.

User

This part of the section contains details regarding the user who is logged-in. This includes information such as the name of the user currently logged-in, other user type(s) he/she can access, other system connected with HRMIS and the logout option. The User is located at the upper-right side of the screen (See Figure 3-32).

Figure 3-32 User Form

1. To logout from the system, click on the Down icon beside the name of the user.

2. Click the Logout tab, the user will be directed to the main page of HRMIS.
