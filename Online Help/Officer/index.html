Login

To login to the system, the user is required to enter a valid username and password on the Login Form at the center of the Main Page (See Figure 1-1).

Figure 1-1 DOST-CO HRMIS Main Page

How to Login

1. Enter Username and Password on the entry boxes labeled respectively. Then click Login button (See Figure 1-2).

Figure 1-2 Login Form

2. If login is successful, you will be directed to the Default Screen.

Default Screen

Once logged in to the system you will be automatically directed to the PDS Section (See Figure 2).

Figure 2 Default Page

Attendance

Click the Attendance from the left navigation bar to view the Three king of Attendance (See Figure 3-16).

Figure 3-15 Daily Time Record page

Note:Default DTR shown is of the current month.

2.To view attendance from previous month(s), specify Month and Year desired from the drop-down lists.

Request


This section allows the employees to send request files on-line. This is where the employee can file for Official Business Request, file for Seminars/Trainings attendance, file for Leave application, file for Travel Order authority, update employee’s Personal Data Sheet, request for reports generation and monetization of leave benefits.

Click the Request from the left navigation bar to view the Drop list of Request (See Figure 3-17).

Official Business


1. Click the Request from the left navigation bar and click Official Business to view the Official Business window (See Figure 3-11).

Figure 3-20 Official Business page

2. Click the Add New Request and type in information needed such as Request Date, Date from, Date to, Time from, Time to, Destination, Purpose. and Attach File.

3. When done, click Submit button to proceed the Request, Click Cancel to Cancel the operation.

4. To print a copy of request, click Print/Preview button . The printable document will appear in a new window.

Note: To view the list of approved Official Business, click the List of Approved OB link.

Leave


1. Click the Request from the left navigation bar and click Leave to view the Leave window (See Figure 3-11).

2. Click the Add New Request and Specify the Leave Type from the drop-down list.

Note: To view the list of approved OB, click the List of Approved OB link.
Note: Only Regular Employees have Leave credits.

Travel Order


1. Click the Request from the left navigation bar and click Travel Order to view the Travel Order window (See Figure 3-23).

2. Click the Add New Request and type in information needed such as Destination, Date from, Date to, Purpose. and Attach File.

4. To print a copy of request, click Print/Preview button . The printable document will appear in a new window.

DOST Online Help

User

This part of the section contains details regarding the user who is logged-in. This includes information such as the name of the user currently logged-in, other user type(s) he/she can access, other system connected with HRMIS, other servers connected with HRMIS and the logout option. The User is located at the upper-right side of the screen (See Figure 3-58).

Figure 3-58 User Form

How to Switch to ERMS

1. To switch from HRMIS to ERMS, click on the Down icon beside the name of the user.

2. Click the ERMS link, the user will be directed to the default screen of ERMS.

Note:The ERMS link is only available if there is DOST-CO ERMS installed in the system.

How to Switch to PMS

1. To switch from HRMIS to PMS, click on the Down icon beside the name of the user.

2. Click the PMS link, the user will be directed to the default screen of PMS.

Note:The PMS link is only available if there is DOST-CO PMS installed in the system.

Update PDS Request

1. To logout from the system, click on the Down icon beside the name of the user.

2. Click the Logout link, the user will be directed to the main page of HRMIS.
