1. Click the Add Employee link on the center of 201 File page to view the Add New Employee page.

201 File page

2. To add new employee, fill-up the entry boxes that constitute the parameters for adding new employee.

Note: The Surname, First Name and Middle Name of the employee are required in order to add new employee.

Add Employee form

3. When done click Add button to save employee’s information or the Clear button to reset parameters.

There are two ways to browse employee records. First using the filter records and second using the search engine.

1. To select all employees using the filter records, select from the dropdown list of the maximum number of employees.

201 File page

2. To select a specific employee using the search engine, type in the employee’s number or name in the Search textbox found in the right side of the screen.

1. To add user account, click User Account on the dropdown list of Libraries tab.

2. Specify the Access Level from the drop-down list.

3. For HR Module, fill-up the entry boxes that constitute the parameters for adding new User Account.

HR Module User Account form

4. Place a tick mark on the Assistant check box if the employee is assigned as HR Assistant then place a tick mark on the check box of the section(s) that the employee can access.

5. Place a tick mark on the HR Officer check box if the employee is assigned as HR Officer then place a tick mark on the all section check box to access all sections.


6. For Finance/Payroll Module, fill-up the entry boxes that constitute the parameters for adding new User Account.

Payroll Module User Account form

7. Place a tick mark on the Assistant check box if the employee is assigned as Cashier Assistant then place a tick mark on the check box of the section(s) that the employee can access.

8. Specify the Payroll Group assigned to the Cashier Assistant from the drop-down list.

9. Place a tick mark on the Finance Officer check box if the employee is assigned as Finance Officer then place a tick mark on the all section check box to access all sections.


10. For Employee Module, Officer Module and Execom Module, fill-up the entry boxes that constitute the parameters for adding new User Account.

Employee Module User Account form

11. Specify the Employee Name from the drop-down list.

12. Type in the Username and Password on the entry boxes provided.

13. When done click Add button to add new User Account or the Clear button to reset parameters.
